# Turn your 2D design into a 3D printed object ## Goal Creating 2D images is many times easier and more intuitive than controlling 3D. Today, we will work on many different techniques and softwares to turn 2D images into 3D geometry. Through this process, you can add original and creative manual touches to your final 3D model. However we will not focus on specific 2D image techniques; I will leave that to your own unique artistic practice. ## 2D images A **bitmap** is a representation of an image, consisting of rows and columns of pixels, that each store color information. Each pixel (picture element) contains a color value for a number of channels – usually red, green, and blue. When you view these channels together, at a high enough resolution, all of the different colored pixels form a complete image. These images can then be output to video, film, print, etc. **Vector** images are not based on pixel patterns, but instead use mathematical formulas to draw lines and curves that can be combined to create an image from geometric objects such as lines, circles, triangles, squares, and other polygons. Vector images are edited by manipulating the lines and curves that make up the image using a program such as Adobe Illustrator. Bitmap formats are best for images that need to have a wide range of color gradations, such as most photographs. Vector formats, on the other hand, are better for images that consist of a few areas of solid color. Examples of images that are well suited for the vector format include logos and text. Black and white silhouette designs work best. ### How: - Prepare .svg silhouette vector image. - From Blender, import the image. ![2d3d shape bl00](img/07/Chandelier.svg) The source vector image as .svg. ![2d3d shape bl](img/07/bl01.png) - Choose the imported object and scale it up. - From the right tool settings, choose the curve and add the Extrude value.
